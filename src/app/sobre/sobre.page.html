Sobre Quem somos nós? Projeto de Gamificação para o Ensino de Algotimos
O embrião do projeto de extensão "Gamificação para o Ensino de Algoritmos" nasceu no ano de 2017, no IFPE Campus Garanhuns, a partir de uma integração entre as disciplinas de Educação Física e Algoritmos. Neste ano, foi desenvolvido um jogo de tabuleiro, intitulado “Java Project”, cujo principal objetivo era abordar a lógica de programação com uma nova "roupagem", visto os altos índices de reprovação na disciplina de Algoritmos.
Com o resultado positivo que foi recebido, percebeu-se que a proposta poderia ser amadurecida e se tornar um projeto no Campus, e assim foi feito.
O projeto foi realizado em 2018 e a equipe reformulou todo o funcionamento e a lógica do primeiro tabuleiro e produziu um novo, intitulado “Programming”, que contava agora com três modalidades que poderiam ser adaptadas no mesmo tabuleiro: uma que somente utilizava lógica de programação (intitulada “Arbitrária/Wally”), e outras duas com linguagem técnica: uma na linguagem Java e outra pseudolinguagem Portugol. Os resultados também foram muito positivos, tanto com o público interno, quanto o externo.
Em 2019 o projeto foi renovado, trazendo agora uma nova proposta: o desenvolvimento de um aplicativo móvel (este que você está usando!) que pudesse utilizar os mesmos conceitos dos jogos desenvolvidos anteriormente e também ensinar e estimular o aprendizado em lógica de programação.
No decorrer dos projetos, a equipe desenvolveu toda uma plataforma, nomeada “Se Liga Aê!”, que tem como objetivo acompanhar todo o processo de ensino-aprendizagem. Como se trata de uma plataforma, ela, além de incluir o jogo de tabuleiro Programming, também possui materiais de estudo, como algumas apostilas, calendários, folhas de planejamento, etc.
